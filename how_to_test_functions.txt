NOTES ON HOW TO GENERATE AND COMPARE TEST IMAGES FOR NEW FUNCTIONS:

How to generate a 1-D image containing a function's profile.
	-- "centered" = function centered (r=0) at x = 50
	-- "onesided" = function centered at x = 1


Example: Sersic function

Edit config_functest_centered_template.dat and config_functest_onesided_template.dat to
have correct input parameters for function

Then run makeimage:
e.g., with Sersic function:

./makeimage config_functest_centered_sersic.dat -o functest_1dcentered_sersic.fits --ncols 100 --nrows 1
./makeimage config_functest_onesided_sersic.dat -o functest_1d_sersic.fits --ncols 100 --nrows 1


In Python:

fimfit = "/Users/erwin/coding/imfit/"

# get centered profile
imcent = pyfits.getdata(fimfit + "functest_1dcentered_sersic.fits")
rrcent = N.arange(1.0, 101.0)
clf();plot(rrcent, imcent[0,:])

# get one-sided profile
im = pyfits.getdata(fimfit + "functest_1d_sersic.fits")
rr = N.arange(0.0, 100.0)
clf();plot(rr, im[0,:])
